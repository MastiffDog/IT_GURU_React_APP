# Описание проекта
Использован стандрартный шаблон для проекта, который включает в себя:

- react 18
- typescript
- storybook
- jest
- eslint
- prettier
- husky
- zustand

Реализован функционал ТЗ, настройки по storybook записаны на будущее, т.к. когда я разворачиваю проект,
я сразу настраиваю webpack и структуру папок для максимального удобства на будущее. 

Проект реализован в соответствии с методологией feature sliced design.

По стилям - модульный подход с добавлением хешей в классам для изоляции (+ кастомные компоненты)

API DummyJSON не поддерживает проверку токена на сервере, поэтому реализуем простейший вариант сессионного управления, основанный на успешном получении токена при входе

В макет страницы продуктов в навбар добавлена кнопка "Выйти" - для того, чтобы можно было полноценно 
протестировать процедуру входа-выхода
